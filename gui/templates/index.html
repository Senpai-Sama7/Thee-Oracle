Help & Documentation

Learn how to use Oracle Agent effectively

🚀 Quick Start

1. Start a Conversation

Type your question or request in the chat box at the bottom and press Enter. Oracle Agent will respond and can help you with:

  • Answering questions and providing information
  • Executing commands and scripts
  • Managing files and directories
  • Making web requests and API calls
  • Analyzing data and generating reports

2. Use Tools Directly

Click on tools in the sidebar to execute them manually:

  • Shell: Run terminal commands
  • Files: Read, write, list, or delete files
  • HTTP: Make web requests to APIs
  • Vision: Capture screenshots for analysis

3. Configure Your Experience

Visit Settings to customize:

  • AI model and creativity level
  • Security settings and timeouts
  • Cloud backup preferences
  • Advanced debugging options

✨ Features Guide

🤖 AI Conversations

Oracle Agent uses advanced AI to understand and respond to your requests. It can:

  • Understand natural language requests
  • Execute tools to accomplish tasks
  • Remember context throughout conversations
  • Learn from your preferences
Example: "List the files in the current directory and tell me which ones are Python files"

🔧 Tool Execution

Oracle Agent can execute various tools safely:

  • Shell Commands: Run terminal commands with output capture
  • File Operations: Read, write, organize files and folders
  • Web Requests: Access APIs and fetch web content
  • Vision Analysis: Analyze screenshots and images
Example: "Create a backup of all .py files in a compressed archive"

💾 Data Management

Your data is handled securely:

  • Conversations saved locally in SQLite database
  • Optional cloud backup to Google Cloud Storage
  • File operations restricted to project directory
  • All commands executed in sandboxed environment

⚙️ Settings Explained

AI Model Settings

  • Model Selection: Choose between fast (Flash) or advanced (Pro) models
  • Conversation Length: Maximum turns before conversation ends
  • Creativity Level: Control response creativity (0% = factual, 100% = creative)

Security Settings

  • Timeouts: Prevent long-running commands from hanging
  • File Sandbox: Restrict file operations to safe areas
  • Command Validation: Validate commands before execution

Cloud Settings

  • GCS Backup: Store conversations in Google Cloud Storage
  • Auto-backup: Automatically save after conversations
  • Bucket Configuration: Choose your storage bucket

🔧 Troubleshooting

Common Issues

Problem: "Agent not responding"
Solution: Check if the agent is initialized (green status dot) and try refreshing the page
Problem: "Tool execution failed"
Solution: Check tool permissions and ensure commands are valid
Problem: "GCS backup not working"
Solution: Configure Google Cloud credentials and bucket name

Performance Tips

  • Use Gemini Flash for faster responses
  • Limit conversation length for complex tasks
  • Clear history periodically to improve performance
  • Use appropriate timeouts for long-running operations
